# TimeSheet

    This is a sample applicaion with react and material-UI

# Built with:

Node.js,
React,
Material-UI

# Prerequisites

Node.js >= 10

# Installation

1. git clone git@github.com:abbas565/timesheet.git
2. cd quizBuilder
3. npm install
4. Run Application with npm start

Open http://localhost:3000 to see the app.
